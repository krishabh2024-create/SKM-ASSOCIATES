One-time setup needed to enable sync between your phone and laptop. Takes about 5 minutes.

1
Open console.firebase.google.com and sign in with your Google account.
2
Click Add project → name it anything (e.g. skm-manager) → Continue → Disable Analytics → Create project.
3
In the left menu click Build → Realtime Database → Create database → choose Start in test mode → Enable.

Copy the URL shown (e.g. https://skm-xxxx-default-rtdb.firebaseio.com).
4
Go to ⚙️ Project Settings (gear icon top-left) → scroll down → click </> Web icon → register app → copy the apiKey value.
5
Paste both values below and tap Connect & Start. Save the updated file and open it on all your devices.
Your data is stored in YOUR own Firebase account.
